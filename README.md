# 🃏 小丑牌

Balatro 风格的「扑克 + 小丑牌组合」单机网页游戏。从 0 金币、0 张小丑牌起步，出 5 张以内的扑克牌型 + 小丑牌加成攒分，4 手内打到盲注目标分通关，进商店买卡变强，依次推进 3 个递增盲注关卡。

- **玩法循环**：选牌 → 出牌 → 触发 Joker → 计分 → 通关 → 进商店 → 买卡 → 打下一关
- **3 关盲注**：小盲注 300 / 中盲注 500 / 大盲注 800，全过即通关
- **6 张 Joker 候选库**、5 槽上限，商店每次随机抽 3 张
- 完整动效（飞牌 / 飞字 / Joker 金光 / 计分爆字）、设置面板、本地启发式 AI 出牌
- 纯前端，无后端、无登录、无 API key

技术栈：Vue 3 + Vite，纯 CSS 动画。

## 安装

```bash
npm install
```

需要 Node.js（建议 18+）。

## 使用

```bash
npm run dev       # 启动开发服务器，默认 http://localhost:5173/
npm run build     # 生产构建，产物输出到 dist/
npm run preview   # 本地预览构建产物
```

打开浏览器访问开发服务器地址即可开玩：点手牌选 1–5 张后「出牌」，或点「🤖 AI 出牌」自动选最优组合；右上角 ⚙️ 可调动画速度等设置。
